Description [src]

Signs the data using the client self-signed certificate.

You must have called fwupd_client_connect_async() on self before using this method.

Available since: 1.5.0

This method completes asynchronously. Use fwupd_client_self_sign_finish() inside the GAsyncReadyCallback to obtain the result of the operation.
