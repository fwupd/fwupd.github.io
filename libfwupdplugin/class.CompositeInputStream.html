A input stream that is made up of other partial streams, e.g.

[xxxxxxxxxxxx] [yyyyyyyy] | 0x6 | |0x4| \ \ / / \ \ / / \ | | | | | [xxxxxxyyyyyy]

xxx offset: 2, sz: 6 yyy offset: 0, sz: 4
