final class FwupdPlugin.UsbInterface : FwupdPlugin.Firmware
  implements Fwupd.Codec {
  /* No available fields */

This object is a thin glib wrapper around a libusb_interface_descriptor.

All the data is copied when the object is created and the original descriptor can be destroyed any at point.
